What is 2D Rigged Animation

Well first, Rigging is a technique that creates a virtual skeleton that you can move easily. A Rig Skeleton can be used both here and in 3D Animation. To match this in a 2D format, all pieces of a character or moving object will be drawn indivivually. They will then be pieced together creating a skeleton that can control all the pieces at once.
